# Implementing Data Structure using OOP


## How to Use
Include the .hpp files in your main program. For compiling, make sure to link the .hpp files using this command.
```
g++ <file.cpp> -I.
```
For example, 
```
g++ example.cpp -I.
```
You can also check the example.cpp for better example.
